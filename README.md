# BMI Calculator (Python | Tkinter)

A simple **BMI (Body Mass Index) Calculator** built using Python and the Tkinter library.

---

# About the Project
This project provides a simple graphical user interface (GUI) that allows users to calculate their BMI by entering:

- Height (in centimeters)
- Weight (in kilograms)

The application then calculates and displays the BMI value.

---

# BMI Formula
BMI = weight (kg) / (height (m))²

# Features
- Simple and user-friendly GUI
- BMI calculation
- Input validation for incorrect values
- Built with Tkinter (no external libraries required)

---

# Requirements
- Python 3.x  
- Tkinter (included by default with Python)
